# Procedury onboardingu nowego pracownika - TechCorp

## Dzień 1: Powitanie i formalności

### Dział HR
- Podpisanie umowy o pracę i dokumentów NDA
- Wydanie identyfikatora i karty dostępowej
- Omówienie pakietu benefitów (MultiSport, prywatna opieka medyczna Luxmed, ubezpieczenie grupowe)
- Prezentacja kultury organizacyjnej i wartości firmy

### Dział IT
- Wydanie laptopa (MacBook Pro M3 lub Dell XPS 15 - do wyboru)
- Konfiguracja konta firmowego (email @techcorp.pl, Slack, Jira, Confluence)
- Instalacja VPN (Cisco AnyConnect) i konfiguracja 2FA (Google Authenticator)
- Szkolenie z podstaw bezpieczeństwa IT (1 godzina)

## Tydzień 1: Wprowadzenie techniczne

### Zespół deweloperski
- Prezentacja architektury systemu (mikroserwisy na Kubernetes)
- Dostęp do repozytoriów Git (GitHub Enterprise)
- Przegląd pipeline CI/CD (Jenkins + ArgoCD)
- Przypisanie mentora technicznego na okres 3 miesięcy
- Pierwszy task z tablicy Jira (zwykle bug fix lub mały feature)

### Środowisko deweloperskie
- Środowisko lokalne: Docker Compose
- Staging: GKE cluster `staging-eu-west1`
- Produkcja: GKE cluster `prod-eu-west1` (dostęp read-only przez pierwszy miesiąc)

## Miesiąc 1: Integracja

- Spotkanie 1:1 z team leadem co tydzień
- Code review minimum 3 PR innych osób tygodniowo
- Udział w sprint retrospekcji i planning
- Prezentacja "Kim jestem" na spotkaniu zespołu

## Miesiąc 3: Ocena okresu próbnego

- Przegląd celów z mentorem
- Feedback 360° od zespołu
- Decyzja o przedłużeniu/zakończeniu okresu próbnego
- Plan rozwoju na kolejne 6 miesięcy
