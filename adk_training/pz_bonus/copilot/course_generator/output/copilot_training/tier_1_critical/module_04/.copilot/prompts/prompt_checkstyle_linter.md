# Prompt dla uruchamiania Checkstyle
Uruchom narzędzie Checkstyle na wskazanym pliku Java w projekcie.
**Argumenty:**
-   `filePath`: Pełna ścieżka do pliku Java do sprawdzenia (np. `src/main/java/org/springframework/samples/petclinic/owner/Owner.java`).
**Przykład użycia:**
`Copilot, użyj umiejętności runCheckstyle na src/main/java/org/springframework/samples/petclinic/owner/Owner.java.`
`Copilot, uruchom Checkstyle dla tego pliku.` (jeśli plik jest otwarty w edytorze)
