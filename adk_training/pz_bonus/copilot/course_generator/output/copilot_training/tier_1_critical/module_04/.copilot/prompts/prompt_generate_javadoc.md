# Prompt dla generowania Javadoc
Wygeneruj standardowy blok Javadoc dla zaznaczonej sygnatury metody.
**Argumenty:**
-   `methodSignature`: Sygnatura metody (np. `public String getFirstName()`).
**Przykład użycia:**
`Copilot, użyj generateJavadoc dla tej metody.` (po zaznaczeniu sygnatury)
`Copilot, wygeneruj Javadoc dla metody public void saveOwner(Owner owner).`
