# Prompt dla generowania boilerplate testu JUnit 5
Wygeneruj podstawową strukturę testu jednostkowego JUnit 5 dla podanej klasy Java.
**Argumenty:**
-   `className`: Nazwa klasy Java, dla której ma zostać utworzony test (np. `OwnerRepository`).
**Przykład użycia:**
`Copilot, użyj umiejętności generateTestBoilerplate dla OwnerRepository.`
`Copilot, wygeneruj test JUnit dla klasy VetService.`
