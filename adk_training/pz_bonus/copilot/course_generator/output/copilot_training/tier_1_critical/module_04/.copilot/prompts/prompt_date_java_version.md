# Prompt dla wyświetlania informacji o projekcie
Wyświetl aktualną datę lub wersję Javy używaną w projekcie.
**Argumenty:**
-   `infoType`: Typ informacji do wyświetlenia ('date' lub 'java version').
**Przykład użycia:**
`Copilot, użyj showProjectInfo date.`
`Copilot, pokaż mi wersję Javy w projekcie.`
